# Tueban Product Specification

## 1. Product Type

Tueban là nền tảng:

Learning + Proof System

Nó kết hợp:

- LMS
- Progress Tracking
- Learning Ledger

## 2. Core Features

### Learning System

- khóa học
- lesson
- topic
- media

### Assessment

- quiz
- practice
- report

### Progress Tracking

- lesson progress
- topic completion
- course completion

### Proof System

- ledger events
- learning history
- verification

## 3. User Types

Hệ thống có các loại user:

- visitor
- member
- instructor
- admin
- owner

## 4. Key Capabilities

- sequential learning
- proof recording
- audit trail
- secure access

## 5. Platforms

Hệ thống chạy trên:

- web
- cloudflare
- serverless architecture

## 6. Future Expansion

Có thể mở rộng:

- mobile app
- AI learning assistant
- public proof explorer
